PARTE 2 FII: INTRODUCCIÓN AL DESARROLLO WEB

En esta segunda parte del temario nos enfocaremos en cómo es el proceso de creación de una página web. Desde las herramientas de desarrollo, como los IDEs (Entornos de Desarrollo Integrado) y los repositorios (centralizado y distribuido). Hasta los lenguajes utilizados para el desarrollo de las páginas web, los más comúnes son HTML, CSS y JavaScipt (aunque en estos temas solo utilizamos HTML y CSS).

HTML: HyperText Markup Language es un lenguaje de marcado que se utiliza para definir el contenido que incluirá la web (frontend), como por ejemplo los textos, imágenes y enlaces a otras páginas.

CSS: su traducción es Hojas de estilo de cascada, este lenguaje se usa para dar una apariencia más agradable a nuestras páginas, modificando su apartado visual y retocando su diseño.
